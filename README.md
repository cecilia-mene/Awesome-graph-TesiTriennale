# Awesome Graph – Web3 Graph Database Project

Progetto universitario sviluppato nell’ambito della mia tesi triennale sul tema dei Graph Database applicati al Web3.

## Obiettivo del progetto

L’obiettivo era creare un sistema in grado di analizzare wallet, NFT e transazioni blockchain utilizzando un database a grafo Neo4j e API sviluppate con FastAPI.

## Tecnologie utilizzate

- Python
- FastAPI
- Neo4j
- Docker
- REST API

## Il mio contributo

Mi sono occupata principalmente dello sviluppo e della modifica di:
- wallet analysis
- transaction analysis
- NFT routes
- operations management

con implementazione di endpoint API e query per l’analisi dei dati blockchain.

## Funzionalità principali

- Analisi wallet
- Gestione transazioni
- Analisi NFT
- Query su database a grafo
- API REST con FastAPI
